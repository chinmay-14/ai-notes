AI Notes Manager

Hey! 
This is a full-stack Notes application I built to understand how real-world web apps work — from frontend to backend to database.

The goal was simple: create a place where users can securely store and manage their notes. But while building it, I got hands-on experience with authentication, APIs, and deployment.


##  Features

*  User Signup & Login (secure authentication using JWT)
*  Create, view, and delete notes
*  Notes are stored in a real database (MySQL)
*  Fully deployed (Frontend + Backend)

##  Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express
* **Database:** MySQL (Railway)
* **Authentication:** JWT & Bcrypt
* **Deployment:**

  * Frontend → Vercel
  * Backend → Render

##  Project Structure

ai-notes/
├── frontend/   → React app
└── backend/    → Express server + APIs

##  What I Learned

* How frontend and backend communicate using APIs
* Implementing authentication using JWT
* Working with databases (MySQL)
* Deploying full-stack apps (Vercel + Render)
* Debugging real-world issues (which was a big learning )
  
##  Future Improvements
